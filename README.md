# cl-crud2

## Install

#### npm

```shell
npm i cl-crud2
```

#### yarn

```shell
yarn add cl-crud2
```

#### Browser

```html
<script src="https://cdn.jsdelivr.net/npm/cl-crud2@0.3.3/dist/cl-crud2.min.js"></script>
```
